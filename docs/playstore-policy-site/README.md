# LoversConnect Policy Site

This folder is ready to deploy as a static Netlify site.

## Deploy with Netlify drag and drop

1. Go to Netlify.
2. Choose "Add new site" then "Deploy manually".
3. Drag this folder into Netlify:
   `docs/playstore-policy-site`
4. Netlify will give you a public HTTPS URL.
5. Use the URL in Google Play Console under:
   `Policy and programs` > `App content` > `Privacy policy`

Recommended Play Console URL after deploy:

`https://YOUR-NETLIFY-SITE.netlify.app/privacy-policy`
